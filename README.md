Web-monitor
===========

Website monitoring by http response time and status code.

How to use it?
--------------

1. Edit config.yml
2. Edit urls.txt
3. Run monitoring

    ruby web-monitor.rb

4. Run web-monitor in crontab

Configuration
-------------

You can enter config file as script parameter:

    ruby web-monitor.rb /path/to/some/config-file.yml
